# 扫描效果PDF生成器 - 使用说明

## 📋 软件简介

扫描效果PDF生成器是一款纯浏览器端的PDF和图片处理工具，可以为您的文档添加逼真的扫描效果，包括边缘阴影、轻微旋转、噪点等，让电子文档看起来像真实的扫描件。

**特点：**
- ✅ 纯前端处理，无需网络，保护隐私
- ✅ 支持离线/内网使用
- ✅ 支持PDF和多格式图片
- ✅ 实时预览效果
- ✅ 一键生成下载

---

## 🚀 快速开始

### 1. 打开软件
双击 `index.html` 文件，在浏览器中打开。

**支持的浏览器：**
- Chrome / Edge（推荐）
- Firefox
- Safari

### 2. 上传文件
点击上传区域，或拖拽文件到页面中。

**支持的格式：**
- PDF 文件 (.pdf)
- 图片文件 (.jpg, .jpeg, .png, .gif, .bmp, .webp, .svg, .ico, .tiff, .tif, .avif, .heic, .heif)

### 3. 调整扫描效果
在左侧控制面板调整参数：

| 参数 | 说明 | 默认值 |
|-----|------|--------|
| **颜色模式** | 彩色/黑白切换 | 彩色 |
| **亮度** | 调整整体亮度 | 100% |
| **对比度** | 调整对比度 | 100% |
| **模糊** | 模拟扫描模糊度 | 30% |
| **旋转角度** | 页面整体旋转 | 1° |
| **旋转随机度** | 每页随机旋转范围 | 0.5° |
| **噪点** | 添加扫描噪点 | 10% |
| **发黄** | 模拟纸张老化 | 0% |

### 4. 生成并下载
点击 **"生成扫描版PDF"** 按钮，等待处理完成后，按钮变为 **"⬇️ 下载PDF"**，点击即可下载。

## 💡 使用技巧

### 制作逼真的扫描件
1. 设置**旋转角度**为 0.5-2°
2. 设置**模糊**为 20-40%
3. 添加少量**噪点**（10-20%）
4. 如需复古效果，可添加**发黄**

### 批量处理
PDF文件会自动分页处理，每页都会应用扫描效果。

### 预览调整
调整参数后会实时更新预览，可以边调整边查看效果。

## ⚠️ 注意事项

1. **文件大小**：建议上传的文件不超过 50MB，过大的文件可能导致处理缓慢
2. **浏览器内存**：处理大量页面时，请确保电脑有足够的内存
3. **隐私安全**：所有处理都在浏览器本地完成，文件不会上传到任何服务器
4. **兼容性**：部分特殊格式的图片可能无法正常显示

## 🔧 常见问题

### Q: 为什么生成的PDF背景是黑色的？
A: 已修复。现在生成的PDF背景为白色。

### Q: 支持哪些图片格式？
A: 支持几乎所有常见格式：JPG、PNG、GIF、BMP、WebP、SVG、TIFF、AVIF、HEIC等。

### Q: 可以在内网使用吗？
A: 可以。本软件完全离线运行，无需网络连接。

### Q: 处理大文件时浏览器卡住怎么办？
A: 建议分批处理，或降低PDF的页面数量。处理过程中请勿关闭浏览器。
